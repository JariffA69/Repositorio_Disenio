
# Prototipo - MecanicWeb

Aquí se mostrará el prototipo del proyecto MecanicWeb, ya plasmando lo que se contempló en los wireframes y siendo este la forma en la que se puede visualizar nuestra página. Este al ser ya el prototipo buscamos que se vea lo que planificamos en los wireframes, con lo que realizamos; lógicamente en este momento ya se llena con la información y los colores que tendrá la página para poder observar de manera correcta los tipos de optimización o esos últimos detalles que se le pueden realizar a la página.
