# Wireframes — MecanicWeb
Aqui se mostraran los wireframes iniciales del proyecto MecanicWeb, diseñados como parte del proceso de ideación y planificación visual. Los wireframes representan la estructura básica de las pantallas clave de la aplicación web, sin estilos gráficos ni interacciones.
## Herramientas utilizadas

- Figma (modo diseño sin interacciones)
  
## Wireframes
